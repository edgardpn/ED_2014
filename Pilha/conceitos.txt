Lista estática: representado por um vetor;
		tem-se o uso de endereços contíguos;
		a ordem linear é determinada pelos índices do vetor;
		as vezes resulta em maior esforço computacional.

Lista dinâmica: além do dado, os elementos possuem um ponteiro para o próximo elemtento(elementos encadeados);
		tem pelo menos um ponteiro para o início.

Lista homogênea: um elemento de uma lista possui apenas um dado primitivo(ex.: um número).

Lista heterogênea: um elemento da lista contém um dado composto(ex.: dados de um funcionário).
